# 🎓 CodeAlpha Student Grades Manager (Java)

## 🚀 Project Overview
This project is a simple Student Grades Manager developed using Java.
It allows users to input student names and grades, then calculates and displays the average, highest, and lowest scores.

## ✨ Features
- Input student names and grades
- Store data using ArrayList
- Calculate average score
- Find highest score
- Find lowest score
- Display a complete report

## 🛠️ Technologies Used
- Java
- OOP Basics
- ArrayList
- Loops and Conditional Statements

## 📂 Project Structure
com.student
│
└── StudentGradesManager.java

## ▶️ How to Run
1. Open the project in Eclipse / IntelliJ
2. Compile the program
3. Run StudentGradesManager.java
4. Enter student details when prompted
